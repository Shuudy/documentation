# Puis-je héberger plusieurs sites sur la même offre d'hébergement ?

Oui. Il est possible **d'héberger plusieurs sites internet différents** sur l'offre de l'**[hébergeur web
français](https://www.o2switch.fr/hebergement-illimite/)** o2switch. 

L'hébergement est **multisite** et **multidomaine**. Vous pouvez avoir plusieurs sites différents, sur des noms de
domaines différents. 

Pour configurer un site supplémentaire, il suffit de l'ajouter avec l'aide de l'outil [Domaines
Configurés](/cpanel/domaines/configuration-domaine) dans cPanel.

L'outil [domaines supplémentaires / configurés](/cpanel/domaines/configuration-domaine) est l'outil qui permet de faire
le lien entre un nom de domaine et un répertoire, de votre choix, de votre hébergement. En langage technique, il s'agit
d'un **VirtualHost** ou **hôte virtuel**.

Chaque site internet sera bien **indépendant**. Il n'y aura pas de lien d'un site internet à l'autre. Les sites seront
séparés dans des dossiers différents. *(Indépendants mais pas cloisonés, voir l'outil [Gestion de l'univers
web](/cpanel/o2switch/univers-web-sous-comptes) pour ça)*

Il n'y a **pas de limite** au nombre de sites et domaines que vous pouvez ajouter sur une même offre d'hébergement.
