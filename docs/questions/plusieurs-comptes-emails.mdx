# Puis-je créer plusieurs comptes emails sur le même hébergement web ?

Oui, il est possible de **créer plusieurs comptes emails** sur une même offre d'hébergement o2switch. Pour cela, il
suffit de créer tous les comptes emails que vous souhaitez avec l'aide de l'outil [comptes de
messagerie](/cpanel/emails/adresse-email) sur cPanel. 

Les comptes emails ne sont pas limités à votre nom de domaine principal. Vous pouvez **créer des comptes emails sur tous
les domaines** que vous avez configurés sur votre hébergement.

Exemple, si vous avez configurés mon-domaine.com et mon-domaine.fr sur une même offre, vous pouvez avoir des adresses
emails liées à ces deux domaines là. 

Il n'y a **pas de limite** sur le nombre de comptes emails que vous pouvez créer sur une même offre d'hébergement.
