# Procédure de mot de passe perdu pour un hébergement web

Ce tutoriel explique comment lancer la procédure de mot de passe perdu pour votre [hébergement web
cPanel](https://www.o2switch.fr/). 

## Mot de passe perdu cPanel

Pour lancer la procédure, commencez par vous connecter sur votre [espace client
o2switch](https://clients.o2switch.fr/auth/connexion).

Si vous n'avez plus la main sur votre espace client, vous pouvez lancer la [procédure de mot de passe perdu pour 
l'espace client](/espace-client/auth/changer-mot-de-passe).

Une fois connecté, il faut cliquez sur `Gérer mes services` dans le menu à gauche puis vous rendre dans l'onglet 
`Services actifs`. Ou [cliquez sur ce lien](https://clients.o2switch.fr/mes-produits/offres-actives) en étant
préalablement connecté.

Ensuite, il faut cliquer sur les `...` en face de la prestation d'hébergement concernée puis cliquer sur 
`Mot de passe perdu`. 

<Image
    src="/img/docs/guides/cpanel/mot-de-passe-perdu-cpanel.png"
    title="Initier la procédure de mot de passe perdu depuis l'espace client"
    caption="Comment initier la procédure de mot de passe perdu pour cPanel depuis l'espace client"
    alt="Capture d'écran de l'espace client o2switch pour lancer la procédure de mot de passe perdu"
/>

Une autre page s'affiche, il faut confirmer l'opération en cliquant sur `Demander un nouveau mot de passe`.

<Image
    src="/img/docs/guides/cpanel/retrouver-mot-passe-cpanel.png"
    title="Lancement de la procédure de mot de passe perdu cPanel"
    caption="Confirmation du lancement de la procédure de mot de passe perdu pour l'hébergement cPanel"
    alt="Capture d'écran de l'espace client o2switch pour confirmer la procédure de mot de passe perdu"
/>

Le nouveau mot de passe sera envoyé quelque instant plus tard par email à l'adresse indiquée.
