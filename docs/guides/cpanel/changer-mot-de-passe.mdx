#  Comment changer le mot de passe de cPanel ?

Il est possible de **changer le mot de passe de cPanel**, qui correspond également au mot de passe du compte FTP
principal de l'hébergement, pour faire le changement, il faut suivre la procédure suivante : 
  - [Se connecter sur cPanel](/guides/cpanel/comment-se-connecter)
  - Cliquer sur l'identifiant qui se situe tout en haut à droite de l'écran pour faire apparaître un menu déroulant 
  - Cliquer sur le lien `Mot de passe et sécurité` 
  - Ensuite il faut remplir le formulaire en indiquant son ancien mot de passe puis le nouveau

:::info[Mot de passe perdu]
Si vous n'arrivez pas à vous connecter sur cPanel ou que vous avez perdu votre mot de passe, alors vous pouvez
[lancer la procédure de mot de passe perdu](/guides/cpanel/mot-de-passe-perdu) depuis votre espace client o2switch. 
:::
