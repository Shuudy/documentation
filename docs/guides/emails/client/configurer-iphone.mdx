# Configurer une adresse email sur iPhone 

Ce guide explique comment configurer une adresse email, créée sur un [hébergement web
o2switch](https://www.o2switch.fr/hebergement-illimite/) sur un iPhone.

Cela vous permettra de consulter et d'envoyer des emails depuis votre téléphone.

## Prérequis avant la configuration


Dans un premier temps, retrouvez l'email `Bienvenue chez o2switch`. Ce mail contient une rubrique `Créer vos emails` qui
donne toute la configuration email et notamment l'adresse du serveur de la forme `qqch.o2switch.net` qui sera nécessaire
plus tard.

## Application Mail par défaut 

Si vous souhaitez configurer votre adresse email sur l'application  **Mail** par défaut de votre iPhone, alors il faut :

<Badge text="1" type="danger" /> Accédez à `Réglages`. Si besoin, vous pouvez retrouver les réglages en effectuant 
une recherche dans la bibliothèque des applications sur le terme `Réglage`.

<Badge text="2" type="danger" /> Naviguez dans `Apps` 

<Badge text="3" type="danger" /> Ensuite touchez l'application `Mail`. 

<Badge text="4" type="danger" /> Puis cliquez sur `Comptes Mail`

<Image
    src="/img/docs/guides/emails/client/o2switch-configurer-email-iphone-1.png"
    title="Première étape de la configuration d'une adresse email sur iPhone"
    caption="Première étape dans la configuration de l'adresse email o2switch sur iPhone"
    alt="Capture d'écran des premiers écrans de configuration d'une adresse email sur iPhone"
/>

<Badge text="5" type="danger" /> Ensuite choisissez `Autre`

<Badge text="6" type="danger" /> Puis `Ajouter un compte Mail`

<Badge text="7" type="danger" /> Sur l'écran suivant, renseignez les éléments suivants : 
- Le `Nom` va correspondre au nom d'affichage des emails que vous allez envoyer, vous pouvez mettre votre nom/prénom
- L'`adresse` correspond à l'adresse email 
- Le `Mot de passe` correspond au mot de passe de votre adresse email. Donc ce que vous avez renseigné dans l'[outil
Compte de messageries](/cpanel/emails/adresse-email) lors de la création du compte
- Pour la `Description`, cela va correspond au nom d'affichage sur votre téléphone plus tard, vous pouvez mettre ce que
vous voulez 

<Badge text="8" type="danger" /> Sur l'écran suivant, il faut configurer le serveur d'envoi et le serveur de réception.

Assurez-vous d'être en l'onglet `IMAP`. Il est normalement sélectionné par défaut.

Pour le serveur de réception : 
- `Nom d'hôte` : reprenez le nom du serveur communiqué dans le mail `Bienvenue chez o2switch`. C'est de la forme 
`qqch.o2switch.net`
- `Nom d'utilisateur` : votre adresse email complète 
- `Mot de passe` : le mot de passe de l'adresse email 

Pour le serveur d'envoi, ce sont les mêmes réglages finalement : 
- `Nom d'hôte` : reprenez le nom du serveur communiqué dans le mail `Bienvenue chez o2switch`. C'est de la forme 
`qqch.o2switch.net`
- `Nom d'utilisateur` : votre adresse email complète 
- `Mot de passe` : le mot de passe de l'adresse email 

Ensuite vous pouvez cliquer `Suivant` en haut à droite. Cela lancera une phase de vérification qui peut prendre quelques
secondes.

<Image
    src="/img/docs/guides/emails/client/o2switch-configurer-email-iphone-2.png"
    title="Deuxième étape de la configuration d'une adresse email sur iPhone"
    caption="Deuxième étape dans la configuration de l'adresse email o2switch sur iPhone"
    alt="Capture d'écran des écrans de configuration d'une adresse email sur iPhone"
/>

<Badge text="9" type="danger" /> Enfin, il ne reste plus qu'à cliquer sur `Enregistrer` pour finaliser la configuration
de l'adresse email.

L'adresse email s'affichera dans les comptes emails enregistrés sur l'iPhone.

## Application Gmail iPhone 

Si vous souhaitez configurer votre adresse email sur l'application  **Mail** par défaut de votre iPhone, alors il faut :

<Badge text="1" type="warning" /> Ouvrez l'application `Gmail`. Si besoin, vous pouvez la retrouver en effectuant une
recherche dans la bibliothèque des applications sur le terme `Gmail`.

<Badge text="2" type="warning" /> Ensuite, cliquez sur votre avatar pour ajouter un compte. Si vous n'avez pas de 
comptes emails configurés, un assistant s'affichera à la place et proposera directement d'ajouter un compte email. 

<Badge text="3" type="warning" /> Cliquez sur `Ajouter un compte`.

<Badge text="4" type="warning" /> Cliquez sur `Autre (IMAP)`

<Badge text="5" type="warning" /> Notez votre adresse email dans le champs `Adresse e-mail`

<Image
    src="/img/docs/guides/emails/client/o2switch-email-iphone-gmail-1.png"
    title="Première étape de la configuration d'une adresse email sur Gmail sur iPhone"
    caption="Première étape dans la configuration de l'adresse email o2switch sur Gmail sur iPhone"
    alt="Capture d'écran des premiers écrans de configuration d'une adresse email sur Gmail sur iPhone"
/>

<Badge text="6" type="warning" /> Dans l'écran des `Paramètres du serveur entrant`, renseignez les valeurs suivantes :
- `Nom d'utilisateur` votre adresse email complète 
- `Mot de passe` celui de votre adresse email. Donc ce que vous avez renseigné dans l'[outil Compte de
  messageries](/cpanel/emails/adresse-email) lors de la création du compte
- `Serveur IMAP` : reprenez le nom du serveur communiqué dans le mail `Bienvenue chez o2switch`. C'est de la forme 
`qqch.o2switch.net`
- `Port` laissez la valeur par défaut 993 
- `Type de sécurité` laissez SSL/TLS 

<Badge text="7" type="warning" /> Dans l'écran des `Paramètres du serveur sortant`, renseignez les valeurs suivantes :
- `Nom d'utilisateur` votre adresse email complète 
- `Mot de passe` celui de votre adresse email. 
- `Serveur IMAP` : reprenez le nom du serveur communiqué dans le mail `Bienvenue chez o2switch`. C'est de la forme 
`qqch.o2switch.net`
- `Port` laissez la valeur par défaut 465 
- `Type de sécurité` laissez SSL/TLS 

<Badge text="8" type="warning" /> Ensuite ajoutez votre nom. Ce nom s'affichera dans les messages que vous envoyez

<Image
    src="/img/docs/guides/emails/client/o2switch-email-iphone-gmail-2.png"
    title="Dernière étape de la configuration d'une adresse email sur Gmail sur iPhone"
    caption="Dernière étape dans la configuration de l'adresse email o2switch sur Gmail sur iPhone"
    alt="Capture d'écran des derniers écrans de configuration d'une adresse email sur Gmail sur iPhone"
/>

C'est terminé, le compte email est configuré, vous pouvez envoyer et recevoir des emails via l'application Gmail sur
votre téléphone.
