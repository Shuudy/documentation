# Configurer une adresse email sur smartphone Android

Ce guide explique comment configurer une adresse email, créée sur un [hébergement
o2switch](https://www.o2switch.fr/hebergement-illimite/) sur un smartphone Android avec l'application Gmail native.

Cela vous permettra de consulter et d'envoyer des emails depuis votre téléphone.


## Prérequis avant la configuration


Dans un premier temps, retrouvez l'email `Bienvenue chez o2switch`. Ce mail contient une rubrique `Créer vos emails` qui
donne toute la configuration email et notamment l'adresse du serveur de la forme `qqch.o2switch.net` qui sera nécessaire
plus tard.

## Configurer l'application Gmail sur Android

Si vous souhaitez configurer votre adresse email sur l'application  **Gmail** par défaut sur Android, alors il faut :

<Badge text="1" type="danger" /> Ouvrez l'application `Gmail`. 

<Badge text="2" type="danger" /> Ensuite, cliquez sur votre avatar puis sur `Ajouter un compte`.

Si vous n'avez pas de comptes emails configurés, un assistant s'affichera à la place et proposera directement d'ajouter
un compte email.

<Badge text="3" type="danger" /> Cliquez sur `Autre` pour le type de compte à configurer.

<Badge text="4" type="danger" /> Remplissez le champs `Saisissez votre adresse email` en notant votre adresse complète.

<Badge text="5" type="danger" /> A la question `De quel type de compte s'agit-il ?` choisissez un compte `IMAP`.

<Image
    src="/img/docs/guides/emails/client/o2switch-configurer-android-1.png"
    title="Première étape de la configuration d'une adresse email sur Gmail sur Android"
    caption="Première étape dans la configuration de l'adresse email o2switch sur Gmail sur Android"

<Badge text="6" type="danger" /> Dans le champs `Mot de passe`, notez le mot de passe de votre adresse email.

Vous avez choisi ce mot de passe lors de la création du compte email avec l'[outil Compte de
messageries](/cpanel/emails/adresse-email)

<Badge text="7" type="danger" /> Les `Paramètres du serveur entrant` devraient être détectés automatiquement. 

Si les paramètres ne se détectent pas automatique, ou si vous avez une erreur de validation, ou une erreur en rapport
avec un certificat SSL invalide, alors vous pouvez renseignez les réglages suivants : 
- `Nom d'utilisateur` : notez votre adresse email complète 
- `Mot de passe` : celui de l'adresse email
- `Serveur` : reprenez le nom du serveur communiqué dans le mail `Bienvenue chez o2switch`. C'est de la forme 
`qqch.o2switch.net`

<Badge text="8" type="danger" /> Les `Paramètres du serveur sortant` devraient être détectés automatiquement. 

Si les paramètres ne se détectent pas automatique, ou si vous avez une erreur de validation, ou une erreur en rapport
avec un certificat SSL invalide, alors vous pouvez renseignez les réglages suivants : 
- `Nom d'utilisateur` : notez votre adresse email complète 
- `Mot de passe` : celui de l'adresse email
- `Serveur` : reprenez le nom du serveur communiqué dans le mail `Bienvenue chez o2switch`. C'est de la forme 
`qqch.o2switch.net`

<Badge text="9" type="danger" /> Sur l'écran des `Options du compte`, vous pouvez laisser les réglages par défaut
puis valider. 

<Badge text="10" type="danger" /> Votre compte de messagerie est configuré, vous n'avez plus qu'à renseigner un 
nom d'affichage. Ce nom s'affichera dans les emails que vous envoyez. 

Vous pouvez maintenant envoyer et consulter vos emails depuis votre téléphone.
