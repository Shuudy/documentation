adresse email créée sur un [hébergement web o2switch](https://www.o2switch.fr).

1 : Sur les connexions non chiffrées, vous pouvez utiliser votre nom de domaine. Le nom de domaine doit être
noté sans les `www.`, ni le `http://` devant. 

2 : Le nom du serveur est de la forme suivante `nom-serveur.o2switch.net`. Ce nom de serveur peut être 
retrouvé à deux endroits : 
- dans le mail `Bienvenue chez o2switch` qui est envoyé à l'ouverture de votre compte d'hébergement 
- dans votre espace client, dans le détail de votre prestation d'hébergement 

A noter que vous pouvez tout de même utiliser votre nom de domaine dans une connexion chiffrée. Mais il faut au
préalable [générer et installer un certificat SSL](/cpanel/securite/lets-encrypt-ssl-gratuit), sinon les logiciels de
messageries retourneront des erreurs en rapport avec les certificats SSL.

3 : Pour le port sortant, en mode de connexion non chiffrée, vous avez le choix entre le port 25 et 26. Si cela
ne fonctionne pas avec le port 25 par défaut, vous pouvez mettre le 26. Certains parefeu ou fournisseur d'accès internet
bloquant le port 25 pour éviter l'envoi de spam.
