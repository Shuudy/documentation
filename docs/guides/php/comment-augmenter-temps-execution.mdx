Sur [les hébergements web o2switch](https://www.o2switch.fr/hebergement-illimite/) le temps maximum durant lequel un
script PHP peut s'exécuter est défini par la valeur du `max_execution_time` dans la configuration de PHP, cette valeur
de 60 secondes par défaut et peut être augmentée en suivant les étapes suivantes : 


<Badge text="1" type="danger" /> [Se connecter sur l'hébergement](/guides/cpanel/comment-se-connecter)

<Badge text="2" type="danger" /> Naviguer dans l'outil [Sélectionner une version de
PHP](/cpanel/logiciels/hebergement-php-multi-version) présent dans la partie des `logiciels` 

<Badge text="3" type="danger" /> Choisir une version personnalisée de PHP (différente de la version dîtes `native`)

<Badge text="4" type="danger" /> Cliquer sur le bouton `use defaults` pour recocher les modules PHP nécessaires 

<Badge text="5" type="danger" /> Aller dans `switch to php options`

<Badge text="6" type="danger" /> Augmenter la valeur du `max_execution_time` puis valider
