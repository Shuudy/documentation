# Comment augmenter la taille maximale des fichiers envoyés en PHP ?

Par défaut, sur **[l'hébergement o2switch](https://www.o2switch.fr/hebergement-illimite/)**, la taille maximale des
fichiers envoyés en PHP est de 256 Mo, cette valeur là peut être augmentée en suivant la procédure suivante : 
  - [Se connecter sur cPanel](/guides/cpanel/comment-se-connecter)
  - Aller dans l'outil [Sélectionner une version de PHP](/cpanel/logiciels/hebergement-php-multi-version) présent dans
    cPanel dans la catégorie des `logiciels` 
  - Définir une version personnalisée de PHP, c'est-à-dire une version différente de la version `native` 
  - Cliquer sur `use defaults` pour recocher tous les modules PHP nécessaires 
  - Cliquer sur le lien `switch to php options` 
  - Augmenter les valeurs suivantes : `upload_max_filesize`, `post_max_size`, `memory_limit`. La plus petite des trois
    valeurs définit la taille d'upload maximale (upload max filesize)
  - Valider les changements

A cet instant, il n'est pas possible d'aller au delà de 512Mo.

512Mo est la taille d'upload maximale des fichiers traités en PHP chez o2switch. Au delà, ça bloquera au niveau de PHP
car ce n'est pas possible d'aller au delà de cette valeur, mais également au niveau du serveur web.
