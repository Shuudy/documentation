# Comment activer le module PHP SOAP sur l'hébergement ?


Par défaut, [sur votre espace d'hébergement web](https://www.o2switch.fr/hebergement-illimite/), le module SOAP n'est
pas chargé dans PHP, il est possible de l'activer depuis cPanel en suivant la procédure suivante : 
  - [Se connecter sur cPanel](/guides/cpanel/comment-se-connecter)
  - Se rendre dans l'outil [Sélectionner une version de PHP](/cpanel/logiciels/hebergement-php-multi-version) qui est
    visible dans la partie `logiciels` 
  - Choisir une version personnalisée de PHP, c'est-à-dire une des versions sans le `native` écrit à coté 
  - Cliquer sur `use defaults` pour recocher les modules PHP courants
  - S'assurer que le module `soap` est bien coché puis valider
