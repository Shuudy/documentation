# Comment forcer l'affichage des erreurs PHP ?

Par défaut, les erreurs PHP ne sont pas affichées, ni loguées sur les [solutions d'hébergement
o2switch](https://www.o2switch.fr/).

Pour forcer l'affichage des erreurs PHP, il faut : 

<Badge text="1" type="danger" /> [Se connecter sur l'hébergement cPanel](/guides/cpanel/comment-se-connecter)

<Badge text="2" type="danger" />  Naviguer dans l'outil [Sélectionner une version de
PHP](/cpanel/logiciels/hebergement-php-multi-version) présent dans la catégorie des `logiciels`

<Badge text="3" type="danger" />  Choisir une version personnalisée de PHP (c'est-à-dire une version différente de la
version native !)

<Badge text="4" type="danger" />  Cliquer sur le bouton `use default` pour recocher les modules PHP utilisés par défaut 

<Badge text="5" type="danger" />  Cliquer sur `switch to php options` (en haut à droite)

<Badge text="6" type="danger" />  Changer le `display_error` sur `on` puis valider
