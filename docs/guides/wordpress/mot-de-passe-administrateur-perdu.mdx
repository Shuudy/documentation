# Comment retrouver la main sur WordPress sans mot de passe ? 

Si vous avez oublié vos codes d'accès WordPress, vous pouvez retrouver la main sur le site internet facilement et 
rapidement avec l'aide de WPTiger. 

L'[outil WPTiger](/cpanel/o2switch/wptiger/) vous permet :
- de vous connecter rapidement à l'administration du site WordPress en 1 clic sans mot de passe 
- de redéfinir le mot de passe de n'importe quel compte utilisateur, y compris un compte administrateur 
- voire même de créer des comptes utilisateurs / administrateurs supplémentaires 

En résumé, WPTiger vous permet de reprendre le contrôle de votre site, même si vous avez perdu vos codes d'accès.

## Mot de passe perdu WordPress avec WPTiger

Pour reprendre le contrôle de votre site, il faut :

<Badge text="1" /> [Se connecter l'interface d'administration](/guides/cpanel/comment-se-connecter) de l'hébergement

<Badge text="2" /> Naviguer dans l'[outil WPTiger](/cpanel/o2switch/wptiger/)

<Badge text="3" /> Cliquer sur `Gérer le site` en face du site concerné 

<Badge text="4" /> Naviguer dans l'onglet des `Utilisateurs` 

Dans l'outil de gestion des utilisateurs WordPress, vous pouvez : 
- vous **connecter rapidement** à n'importe quel compte en cliquant sur `Se connecter`. Cela créera une session, vous
n'aurez pas besoin de renseigner de mot de passe 
- **changer le mot de passe** de n'importe quel compte. L'ancien mot de passe ne sera pas demandé. Vous pouvez même
**changer le rôle** d'un utilisateur. 
- **créer un nouveau compte utilisateur**. Vous pouvez même recréer un compte administrateur.
