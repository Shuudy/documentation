# NodeJS chez o2switch

Cette section regroupe les tutoriels et les procédures en rapport avec l'installation et le déploiement d'application
NodeJS sur l'[hébergement web o2switch](https://www.o2switch.fr/). 

La page contient également des tutoriels en rapport avec nodeJS, comme par exemple l'utilisation des CLI nodeJS / npm
ou Yarn.

En complément, vous pouvez consulter la [documentation de l'outil Setup NodeJS
App](cpanel/logiciels/hebergement-nodejs-multi-version) qui permet de créer un environnement virtuel nodeJS sur
l'hébergement o2switch.
