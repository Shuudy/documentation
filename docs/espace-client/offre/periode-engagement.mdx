# Durée d'engagement des offres d'hébergement

Les offres d'hébergement peuvent être commandées annuellement. La période minimale de facturation est de 1 an.

Les offres d'hébergement mutualisées ne peuvent pas être commandées avec une facturation mensuelle.

À l'expiration, vous recevrez des emails vous invitant à renouveler. Si vous ne souhaitez plus être hébergé, il suffit
de ne pas donner suite et de vous assurer que vous n'avez pas un renouvellement automatique en place sur votre espace
client.

o2switch a augmenté le délai légal de remboursement "satisfait ou remboursé". Si vous avez commandé une offre qui ne
vous convient pas et qu'il n'est pas possible de trouver une solution avec le support, vous pouvez demander le
remboursement sous 30 jours. Plus d'informations sur [cette page](http://www.o2switch.fr/30jours/).
