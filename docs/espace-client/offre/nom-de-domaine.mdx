# Noms de domaines 

<div class="row">
    <div class="col col--9">
        Sur votre espace client, il est possible de [commander des noms de domaines](https://clients.o2switch.fr/acheter-nom-de-domaine). 
        
        o2switch propose plus de 100 extensions de noms de domaines différentes. Les noms de domaines peuvent être achetés depuis votre espace client. 
    
        Le prix d'un nom de domaine dépend :
        - de l'extension de domaine : un domaine en `.fr` n'est pas facturé le même prix qu'un `.com`
        - de l'opération à réaliser sur le domaine. Il y a un prix pour la création d'un domaine, un autre prix pour son 
        renouvellement, sa restauration *(renouvellement après expiration)*, ou même le trade *(changement de propriétaire)*
    </div>
    <div class="col col--3">
        ![Nom de domaine](/img/docs/espace-client/offre/tigerwww.svg)
    </div>
</div>

Le prix d'un nom de domaine dépend :
- de l'extension de domaine : un domaine en `.fr` n'est pas facturé le même prix qu'un `.com`
- de l'opération à réaliser sur le domaine. Il y a un prix pour la création d'un domaine, un autre prix pour son 
renouvellement, sa restauration *(renouvellement après expiration)*, ou même le trade *(changement de propriétaire)*

Vous pouvez retrouver la liste de toutes les extensions proposées, ainsi que les prix pour chaque opération, sur la 
page suivante : [Prix des domaines](https://clients.o2switch.fr/domaine/prix-domaines)

Les noms de domaines seront réservés avec les coordonnées définis dans l'espace client à l'origine de la commande.
Les noms de domaines vous appartiennent. 

Vous pouvez configurer les domaines que vous avez acheté sur votre hébergement avec l'aide de l'outil 
[Domaines Configurés](/cpanel/domaines/configuration-domaine) qui se trouve dans la catégorie des `Domaines`.

Vous pouvez [changer les serveurs DNS](/nom-de-domaine/changer-dns/o2switch) des domaines commandés depuis 
votre espace client. Cela permet de faire pointer le nom de domaine vers un autre prestataire, de votre choix.

Si vous souhaitez transférer les noms de domaine vers un autre registre, vous pouvez 
[récupérer le code de transfert](/nom-de-domaine/code-transfert/o2switch) depuis votre espace client.
