# Les offres o2switch

o2switch propose l'Offre Unique d'**hébergement web mutualisé** déclinée en 3 versions : Grow, Cloud et Pro. Il s'agit
de l'Offre phare.

Si vous avez déjà une offre d'hébergement chez un autre prestataire et que vous souhaitez migrer chez o2switch, vous
pouvez bénéficier d'un [bonus commercial](https://www.o2switch.fr/changer-hebergeur-web/). 

En complément de l'offre unique d'hébergement mutualisé, o2switch propose l'achat de **noms de domaine**.

Le prix d'un nom de domaine varie en fonction de l'extension, à partir de 8,25 € HT / an. 
Vous retrouverez la liste des [extensions de domaines et prix sur cette
page](https://clients.o2switch.fr/domaine/prix-domaines).

o2switch propose également des serveurs dédiés infogérés.
