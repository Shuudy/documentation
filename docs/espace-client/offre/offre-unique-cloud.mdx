# Offre Unique Cloud 

La version Cloud de l'Offre Unique o2switch est la version milieu de gamme. L'offre est particulièrement performante et
permissive en termes de ressources.

Cette offre convient à des utilisateurs ayant de gros besoins mais n'étant pas encore prêts à passer sur l'[Offre Unique
Pro](/espace-client/offre/offre-unique-pro).

L'[offre unique Cloud](https://www.o2switch.fr/hebergement-illimite/) est facturée :
- **1,76 € HT / mois** la première année pour la création de l'offre d'hébergement. Soit 21,12 € HT la première année.
- Renouvellement pour **1 an à 16 € HT / mois**. Soit 192 € HT pour un renouvellement d'une année.
- Renouvellement pour **2 ans à 12 € HT / mois**. Soit 288 € HT pour un renouvellement de deux ans. Tarif dégressif.
- Renouvellement pour **3 ans à 9 € HT / mois**. Soit 324 € HT pour un renouvellement de trois ans. Tarif dégressif.

En termes de ressources, l'Offre Unique Cloud dispose de :
- **12** threads CPU - Cela correspond à la puissance de calcul de l'hébergement. Les threads CPU permettent d'exécuter
  plusieurs tâches simultanément.
- **48 Go** de mémoire RAM - Cela correspond à la mémoire temporaire (très rapide) utilisée pour les données actives.
- **42 Mo/s** d'I/O - Les I/O correspondent aux accès disque, c'est-à-dire la vitesse à laquelle les données sont lues
  ou écrites sur le disque.

Un **nom de domaine est offert** avec l'offre d'hébergement. Il est possible de choisir parmi les extensions suivantes :
.com, .net, .org, .fr, .be, .eu, .re, .pm, .tf, .wf, .yt.

Les caractéristiques techniques de l'Offre Unique Cloud sont les suivantes :
- L'espace disque est illimité, des disques NVMe (rapides, faible latence) sont utilisés.
- Pas de limite sur le nombre de sites ou noms de domaines qu'il est possible de configurer sur l'offre.
- Bases de données illimitées, pas de limite sur la taille des bases de données.
- Comptes emails illimités. Il est possible de créer des comptes emails associés à tous les domaines configurés sur
  l'hébergement.
- 8 sous-comptes Lune inclus gratuitement. Les sous-comptes permettent d'isoler vos sites sur des comptes cPanel à part
  pour améliorer la sécurité de votre hébergement.
- Transfert mensuel illimité, bande passante non contrôlée.
- Sauvegardes journalières automatiques de 45 jours, incluant les bases de données, les fichiers et les emails.
- La suite de sécurité Imunify360 est disponible, il s'agit d'un antivirus temps réel offrant une protection contre les
  malwares et des scans réguliers.
- Certificats SSL gratuits Let's Encrypt, automatiquement renouvelés.
- Les langages/runtimes suivants sont supportés, vous pouvez choisir entre plusieurs versions de
  [PHP](/cpanel/logiciels/hebergement-php-multi-version), [NodeJS](/cpanel/logiciels/hebergement-nodejs-multi-version),
  [Python](/cpanel/logiciels/hebergement-python-multi-version) ou
  [Ruby](/cpanel/logiciels/hebergement-ruby-multi-version).

Un compte NextCloud collaboratif de 250 Go est mis à disposition de chaque client ayant au moins une offre unique Cloud
ou Pro. Il est possible de créer des sous-comptes sur cette offre collaborative.

Sur l'offre unique Cloud, l'accès au support technique est prioritaire 24/7.

À noter qu'il est possible de [changer d'offre](/espace-client/commande/changer-offre), c'est-à-dire de mettre à niveau
ou de rétrograder.
