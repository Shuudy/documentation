# Offre Unique Grow

La version Grow de l'Offre Unique o2switch est la version d'entrée de gamme. Il s'agit de l'offre la moins chère, mais
elle dispose de ressources conséquentes. Il ne s'agit pas d'une offre low cost à faible performance.

Cette offre peut convenir à la majorité des utilisateurs. L'offre peut servir à héberger un ou plusieurs sites internet.

L'[offre unique Grow](https://www.o2switch.fr/hebergement-illimite/) est facturée :
- **2,80 € HT / mois** la première année pour la création de l'offre d'hébergement. Soit 33,60 € HT la première année.
- Renouvellement pour **1 an à 7 € HT / mois**. Soit 84 € HT pour un renouvellement d'une année.
- Renouvellement pour **2 ans à 6 € HT / mois**. Soit 144 € HT pour un renouvellement de deux ans. Tarif dégressif.
- Renouvellement pour **3 ans à 5 € HT / mois**. Soit 180 € HT pour un renouvellement de trois ans. Tarif dégressif.

En termes de ressources, l'Offre Unique Grow dispose de :
- **8** threads CPU - Cela correspond à la puissance de calcul de l'hébergement. Les threads CPU permettent d'exécuter
  plusieurs tâches simultanément.
- **16 Go** de mémoire RAM - Cela correspond à la mémoire temporaire (très rapide) utilisée pour les données actives.
- **16 Mo/s** d'I/O - Les I/O correspondent aux accès disque, c'est-à-dire la vitesse à laquelle les données sont lues
  ou écrites sur le disque.

Un **nom de domaine est offert** avec l'offre d'hébergement. Il est possible de choisir parmi les extensions suivantes :
.com, .net, .org, .fr, .be, .eu, .re, .pm, .tf, .wf, .yt.

Les caractéristiques techniques de l'Offre Unique Grow sont les suivantes :
- L'espace disque est illimité, des disques NVMe (rapides, faible latence) sont utilisés.
- Pas de limite sur le nombre de sites ou noms de domaine qu'il est possible de configurer sur l'offre.
- Bases de données illimitées, pas de limite sur la taille des bases de données.
- Comptes emails illimités. Il est possible de créer des comptes emails associés à tous les domaines configurés sur
  l'hébergement.
- 4 sous-comptes Lune inclus gratuitement. Les sous-comptes permettent d'isoler vos sites sur des comptes cPanel à part
  pour améliorer la sécurité de votre hébergement.
- Transfert mensuel illimité, bande passante non contrôlée.
- Sauvegardes journalières automatiques de 30 jours, incluant les bases de données, les fichiers et les emails.
- L'outil Imunify AV+ est proposé, il s'agit d'un logiciel antivirus que vous pouvez utiliser pour réaliser des scans de
  votre hébergement.
- Certificats SSL gratuits Let's Encrypt, automatiquement renouvelés.
- Les langages/runtimes suivants sont supportés, vous pouvez choisir entre plusieurs versions de
  [PHP](/cpanel/logiciels/hebergement-php-multi-version), [NodeJS](/cpanel/logiciels/hebergement-nodejs-multi-version),
  [Python](/cpanel/logiciels/hebergement-python-multi-version) ou
  [Ruby](/cpanel/logiciels/hebergement-ruby-multi-version).

Contrairement aux offres Cloud et Pro, l'instance NextCloud de 250 Go n'est pas incluse avec une offre Grow.

Sur l'offre unique Grow, l'accès au support technique est disponible 24/7.

À noter qu'il est possible de [changer d'offre](/espace-client/commande/changer-offre), c'est-à-dire de mettre à niveau
ou de rétrograder.
