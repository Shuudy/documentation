# Création d'un compte client o2switch

Vous pouvez créer un compte client o2switch en cliquant sur le lien suivant : [créer un compte client o2switch](https://clients.o2switch.fr/creer-un-compte).

Il faut remplir le formulaire d'inscription en renseignant tous les champs obligatoires. 

Vous pouvez vous inscrire en tant que : 
  * **particulier**
  * une **entreprise** : la raison sociale sera demandée. Le numéro de Siret et numéro de TVA seront optionnels. 
  * une **association** : il faudra renseigner le nom de l'association dans la raison sociale 
  * si vous êtes une collectivé, vous pouvez choisir de vous inscrire en status entreprise
