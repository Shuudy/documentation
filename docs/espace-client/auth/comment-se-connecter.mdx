# Connexion à l'espace client o2switch

Vous pouvez vous connecter sur l'espace client o2switch depuis [cette page](https://clients.o2switch.fr/auth/connexion).

Pour se connecter, il faudra avoir : 
  * votre identifiant client. Vous avez choisi un identifiant lors de la création de votre compte. 
  * votre mot de passe 

<Image
    src="/img/docs/espace-client/auth/connexion-client.png"
    title="Page de connexion à l'espace client"
    caption="Formulaire de connexion pour l'espace client"
    alt="Capture d'écran du formulaire de connexion à l'espace client"
/>

Si besoin, l'identifiant client est rappelé à la fin de tous les emails qui sont envoyés par o2switch. 

<Image
    src="/img/docs/espace-client/auth/rappel-identifiant-client.png"
    title="L'identifiant client est rappelé à la fin de chaque email"
    caption="L'identifiant de connexion est rappelé à la fin des emails envoyés par o2switch"
    alt="Extrait d'un email envoyé par o2switch mettant en évidence l'identifiant client"
/>

Si vous avez activé la [double authentification](/espace-client/auth/double-authentification) sur votre espace client, 
alors l'écran suivant demandera un code supplémentaire.

Si vous avez choisi la double authentification par email, alors le code sera envoyé à l'adresse de contact.

Sinon, le code devra être récupéré depuis une application comme Google Authenticator.
