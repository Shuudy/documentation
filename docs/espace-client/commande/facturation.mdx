# Facturation

Tous les services ([hébergement web](https://www.o2switch.fr/hebergement-illimite/), nom de domaine) peuvent être réglés
par carte bancaire, chèque, virement ou mandat administratif.

Toutes les factures sont disponibles dans votre interface client. 

## Mode de réglement 

O2switch propose 3 modes de règlements principaux : carte bancaire, chèque ou virement.
Nous acceptons également les mandats administratifs, cependant la démarche est assez particulière.

### Paiement par Carte bancaire

Vous pouvez payer par carte bancaire, directement en utilisant le formulaire de paiement qui s'affiche dans le 
panier d'achat. 

Lorsque vous payez par Carte Bancaire, par défaut, la méthode de paiement sera enregistré pour des paiements futurs et 
pour l'activation du renouvellement automatique de la prestation commandée. 

Vous pouvez cliquer sur le lien `Cliquez-ici pour ne pas ajouter` si vous ne souhaitez pas enregistrer votre carte 
bancaire comme une méthode de paiement réutilisable ainsi que le renouvellement automatique.

Si vous avez plusieurs méthodes de paiement enregistrées, vous pouvez cliquer sur `Changer` pour sélectionner la 
méthode de paiement que vous préférez utiliser.

Vous pouvez également cliquer sur le client `Cliquez-ici` pour payer avec une nouvelle carte bancaire.

Votre banque peut demander un paiement 3D-Secure 2. Après avoir rempli le formulaire de paiement, une page s'affichera
et demandera de confirmer le paiement depuis l'application bancaire installée sur votre téléphone. Ou une autre forme
d'authentification biométrique. 

Si le paiement est refusé, o2switch ne peut rien faire. La transaction s'effectue entre votre banque et la banque 
d'o2switch, nous recevons ensuite une confirmation de paiement, mais n'avons pas de contrôle sur le processus de 
validation du paiement. 

Le paiement est validé automatiquement. Néanmoins, en cas de problème nous pouvons vérifier manuellement. En cas de
doute, ne payez pas une seconde fois, contactez d'abord le support par mail pour vérification.

### Paiement par Paypal

Si vous souhaitez payer avec Paypal, vous allez être directement redirigé vers le site de Paypal qui vous proposera de
réaliser le paiement avec la méthode de votre choix. 

Une fois le paiement fait, vous allez être redirigé sur votre espace client, sur une page qui vous indiquera si le
paiement a été validé correctement. 

### Paiement par Chèque

Vous avez la possibilité de payer par chèque. Cependant, cette méthode rallonge la durée de traitement de votre commande
de quelques jours, celle-ci étant validée à réception de votre chèque bancaire. 

Le chèque devra être établi à l'ordre d'`o2switch` et envoyé à l'adresse suivante :

N'oubliez pas de noter l'identifiant de votre compte ou une autre information nous permettant de vous identifier au dos

### Paiement par Mandat administratif

Si vous souhaitez payer par mandat administratif, la démarche est identique au virement, mais il faut contacter à
l'avance le [support](https://www.o2switch.fr/support-hebergeur/) afin que nous puissions débloquer votre facture et que
nous soyons informés.

Avant de contacter le support technique, téléchargez le bon de commande et vérifier que toutes les informations sont 
correctes. La facture sera émise avec les mêmes informations. 

Cette méthode de paiement est réservée aux Organismes publics et collectivités.

## Récupération des factures

Les factures sont envoyés en pièce jointe des emails envoyés par o2switch après le traitement de la commande.

Les factures sont également récupérables depuis votre espace client, dès que le paiement est validé. Tant que le
paiement n'est pas validé, vous avez accès à un "bon de commande".

Pour récupérer votre facture, vous devez aller dans l'[espace client](https://clients.o2switch.fr/auth/connexion), 
puis dans la  `Facturations & Avoirs`. Toutes les factures s'affichent, vous pouvez cliquer sur l'icône de téléchargement
pour récupérer la facture au format PDF.

Dans la page des factures, vous pouvez : 
  * Cliquer sur l'icône de téléchargement pour récupérer la facture 
  * Cliquer sur le bouton avec les `...` pour afficher le détails de la facture. Une facture peut contenir plusieurs 
  lignes d'achats. Cela fait apparaitre un deuxième bouton de téléchargement également. 
  * Utiliser le formulaire de recherche pour rechercher une facture précise. Vous pouvez faire une recherche sur un 
  identifiant d'hébergement, un nom de domaine, un ID de panier.
  * Utiliser le bouton `Filtrer` pour faire une recherche avancée. Vous pouvez filtrer sur la méthode de paiement ainsi
  que la date du paiement.

## Taux de TVA

Les règles sur la [perception de la TVA sur les services électroniques](https://europa.eu/youreurope/business/taxation/vat/vat-digital-services-moss-scheme/index_fr.htm) 
sont appliquées.

Pour résumer rapidement : 
- le taux de TVA de 20% est appliquée si l'adresse de facturation est en France (Entreprise, particulier, association)
- une exception pour les DOM-TOM, le taux de TVA appliqué est réduit ou dans certains cas, il n'y a pas de TVA 
- si c'est une entreprise, dans l'Union Européenne (hors France) et qu'un numéro de TVA valide est renseigné : pas de
  TVA
- si c'est dans l'Union Européenne, sans numéro de TVA valide (entreprise ou particulier) : le taux du pays concerné est
  appliqué
- hors Union Européenne : pas de TVA 

## Questions fréquentes

### Mon paiement n'est pas validé, que faire ?

Avant de payer une seconde fois, contactez le support pour avoir une confirmation que le paiement n'est effectivement
pas passé et attendez le retour du support avant de lancer la moindre démarche.

### Puis-je renouveler mes offres à l'avance ?

Vous pouvez renouveler vos offres d'hébergement et de domaine à l'avance, le temps restant sur vos offres n'est pas
perdu, car c'est la date d'expiration qui est prise en compte et non la date de paiement. 

Nous vous recommandons de renouveler vos offres à l'avance pour éviter toute coupure suite à un oubli ou si vous êtes
dans l'impossibilité le renouveler J-1 avant la date d'expiration.

### Est-ce possible d'activer le renouvellement automatique ? 

Oui, il est possible d'activer le renouvellement automatique. Pour cela, veuillez consulter la page suivante : 
[gestion du renouvellement automatique chez o2switch](//espace-client/commande/renouvellement-automatique)

### Puis-je demander un remboursement ? 

Oui, dans certains cas, il est possible d'obtenir un remboursement. Pour en savoir plus, veuillez consulter cette page :
[demande de remboursement](/espace-client/commande/remboursement)
