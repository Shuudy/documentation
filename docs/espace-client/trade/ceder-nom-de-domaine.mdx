# Transfert de propriété d'un nom de domaine

Sur votre [espace client o2switch](https://clients.o2switch.fr/auth/connexion), il est possible de réaliser un transfert
de propriété d'une prestation d'hébergement ou d'un nom de domaine. Ce transfert de propriété est appelé un `trade`.

Un `trade` *(ou échange)* correspond au transfert administratif d'une prestation d'hébergement ou d'un nom de domaine
d'un client o2switch A vers un client o2switch B. Cela permet de changer le propriétaire de la prestation. Dans cet
exemple, le client B devient le nouveau propriétaire.

Un `trade` est particulièrement utile pour les webmasters ou créateurs de sites. Si le système des [contacts
techniques](/espace-client/gestion-contact/technique) n'a pas été utilisé, un trade permet à un webmaster de céder la
gestion *(la propriété)* d'une prestation qu'il gère vers son client final, afin qu'il en devienne le nouveau
propriétaire.

## Comment changer le propriétaire d'un nom de domaine ?

Le trade d'un nom de domaine d'un client A vers un client B peut se faire depuis l'espace client. Cette procédure est
payante car le changement de propriétaire du domaine est également effectué chez le registre de domaine.

Il y a plusieurs conditions pour qu'un trade de domaine soit possible :
- Il doit s'agir d'une prestation de domaine seule sur l'espace client. Le domaine ne doit pas être attaché en tant que
  domaine offert à un hébergement.
- Le nom de domaine doit être actif. Il ne doit pas être expiré ou bloqué chez le registre.
- Le nom de domaine doit être réservé depuis plus de 60 jours.
- Le nom de domaine ne doit pas avoir été transféré dans les 60 derniers jours.
- o2switch doit gérer le nom de domaine.

**Trade d'un domaine lié à un hébergement en tant que domaine offert**

Si vous souhaitez transférer un nom de domaine qui est associé à un hébergement au titre du domaine offert, il faut :
- Dissocier le domaine de l'hébergement depuis votre espace client.
- Faire un trade pour la prestation d'hébergement (si jamais vous souhaitez changer le propriétaire de l'hébergement).
- Faire un autre trade pour le nom de domaine dissocié.
- Sur le nouveau compte client, il sera possible de réassocier domaine et hébergement pour continuer de profiter du
domaine offert.

Voici les étapes pour changer le propriétaire d'un nom de domaine :
1. Le propriétaire actuel du domaine doit récupérer le code de transfert depuis son espace client.
2. Le nouveau propriétaire doit initier la demande de transfert depuis son espace client.
3. En fonction de l'extension de domaine, il est parfois nécessaire de valider des emails qui sont envoyés par le
   registre de domaine.

## Comment récupérer le code de transfert ?

Pour récupérer le code de transfert du nom de domaine, le propriétaire du domaine doit se connecter sur son espace
client, puis naviguer dans `Gérer mes services`, cliquer sur les `...` puis `Demander le code de transfert`.

## Comment initier le changement de propriétaire du domaine ?

Pour initier le changement de titulaire d'un nom de domaine, le futur propriétaire doit se connecter à son [espace
client o2switch](https://clients.o2switch.fr/auth/connexion) puis naviguer dans `Gérer mes services` puis `Trade`.

Ensuite, il faut remplir le formulaire `Devenez le nouveau propriétaire d'un nom de domaine` en fournissant le nom de
domaine ainsi que son code de transfert.

<Image
    src="/img/docs/espace-client/trade/initier-trade-domaine.png"
    title="Initier la procédure de trade de domaine"
    caption="Comment initier la procédure de changement de titulaire d'un domaine o2switch ?"
    alt="Capture d'écran de l'espace client o2switch expliquant comment changer le propriétaire d'un nom de domaine"
/>

Suite à cela, vous serez redirigé vers un panier de paiement. L'opération de trade est payante, car le changement de
propriétaire est répliqué chez le registre de domaine. Le prix dépend de l'extension mais dans la plupart des cas, cela
est facturé 3,50 € HT en une fois.

Après le paiement, le trade sera traité côté o2switch. Suite au traitement, il est possible que vous receviez des emails
provenant du registre du domaine, avec des liens, pour confirmer l'opération de changement de titulaire.

Ce mail n'est pas nécessairement envoyé, cela dépend de l'extension de domaine. Certaines extensions demandent cette
validation supplémentaire.
