# Récupérer le code de transfert chez Online.net

Ce guide explique comment récupérer le **code de transfert** (authcode, eppcode) d'un nom de domaine réservé chez
Online.net. Cela permet de transférer la gestion administrative du domaine de Online.net vers
[o2switch](https://www.o2switch.fr/hebergement-illimite/).


Si vous ne savez pas comment fonctionne un transfert de nom de domaine, c'est-à-dire comment transférer la gestion du
domaine d'un registre à un autre, on explique cela plus en détails sur la page [Comment transférer un nom de domaine
?](/nom-de-domaine/transferer-un-domaine).

Dans un premier temps, il faut se connecter sur l'interface client d'online (console) en suivant [ce
lien](https://console.online.net/fr/login). 

<Badge text="1" /> Ensuite, il faut aller dans la gestion des domaines en cliquant sur le lien `Domaine` du menu
supérieur  :

<Image
    src="/img/docs/nom-de-domaine/code-transfert/code-de-transfert-online-1.png"
    title="Console de gestion Online.net"
    caption="Console de gestion Online.net. Il faut naviguer dans l'onglet des domaines"
    alt="Capture d'écran de la console de gestion online.net"
/>

<Badge text="2" /> Ensuite, il faut cliquer sur le lien `configurer le nom de domaine` pour accéder à la page du gestion
du domaine. 

<Image
    src="/img/docs/nom-de-domaine/code-transfert/authcode-online-2.png"
    title="Configuration du nom de domaine chez Online"
    caption="Configuration du domaine chez Online.net pour récupérer le code de transfert"
    alt="Capture d'écran de la console de gestion online.net"
/>

<Badge text="3" /> Sur la page suivante, il faut s'assurer d'être sur l'onglet `status` puis cliquer sur `Autoriser les
transferts et afficher l'authcode`.

<Image
    src="/img/docs/nom-de-domaine/code-transfert/recuperation-code-de-transfert-online.net-3.png"
    title="Débloquer le nom de domaine chez Online"
    caption="Il faut débloquer le nom de domaine avant de récupérer le code de transfert"
    alt="Capture d'écran de la console de gestion online.net pour débloquer le domaine"
/>


<Badge text="4" /> La page s'actualise et au même endroit, le code de transfert s'affiche : 

<Image
    src="/img/docs/nom-de-domaine/code-transfert/recuperer-authcode-online.net-4.png"
    title="Le code de transfert s'affiche"
    caption="Après le déblocage, le code de transfert s'affiche"
    alt="Capture d'écran de la console de gestion online.net pour récupérer le code de transfert"
/>

Vous pouvez renseigner ce code de transfert dans votre espace client o2switch pour initier le transfert de domaine.

Attention aux majuscules/minuscules qui sont importantes.
