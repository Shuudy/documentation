# Récupérer le code de transfert chez OVH

Ce guide explique comment **débloquer** votre nom de domaine et récupérer le **code de transfert** (authcode) lorsque
votre nom de domaine est réservé chez OVH. 

Le fait de débloquer le nom de domaine et obtenir l'authcode permet de transférer le nom de domaine chez un autre
registre de domaine, par exemple [votre hébergement web o2switch](https://www.o2switch.fr).

Si vous ne savez pas comment fonctionne un transfert de nom de domaine, c'est-à-dire comment transférer la gestion du
domaine d'un registre à un autre, on explique cela plus en détails sur la page [Comment transférer un nom de domaine
?](/nom-de-domaine/transferer-un-domaine).

## Procédure pour récupérer le code de transfert OVH

Dans un premier temps, connectez-vous sur l'interface OVH via [ce lien](https://www.ovh.com/auth/).

<Badge text="1" />Une fois connecté, la page d'accueil s'affiche. Depuis cette page, il faut cliquer sur `Web Cloud`
situé dans le menu supérieur. 

Suite à cela, vous êtes redirigé vers une page différentes qui présente les outils en rapport avec les services web de
OVH.

<Badge text="2" /> Ensuite il faut cliquer sur `Noms de domaine` afin de dérouler la liste des domaines que vous avez
chez OVH.

<Badge text="3" /> Puis cliquer sur le nom de domaine 3  que vous souhaitez transférer.

/>

<Badge text="4" /> Ensuite il faut désactiver la `Protection contre le transfert`. Un nouvel écran s'affiche et vous
demande la confirmation, il faut valider pour continuer.

<Image
    src="/img/docs/nom-de-domaine/code-transfert/recuperer-authcode-ovh-2.png"
    title="Désactiver la protection des transferts de domaine chez OVH"
    caption="Débloquer le nom de domaine chez OVH"
    alt="Capture d'écran de l'interface de gestion OVH pour débloquer le nom de domaine"
/>

<Badge text="5" />  Ensuite il faut **patienter quelques minutes** puis retourner sur cette page de gestion du nom de
domaine (`Web Cloud` > `Noms de domaines` > Cliquez sur votre domaine). 

<Badge text="6" /> L'affichage changera et un lien `AUTH/INFO` s'affichera. Il faut cliquer sur `AUTH/INFO` puis noter
le code de transfert qui s'affiche dans la fenêtre suivante. 

<Image
    src="/img/docs/nom-de-domaine/code-transfert/transfert-domaine-sortant-ovh-3.png"
    title="Récupération du code de transfert chez OVH"
    caption="Récupération du code de transfert (authcode) chez OVH"
    alt="Capture d'écran de l'interface de gestion OVH pour récupérer le code de transfert"
/>


Vous pouvez ensuite lancer le transfert du domaine depuis votre espace client. Il faut aller dans la rubrique `Gérer mes
services` puis cliquer sur `Détails` en face de la prestation concernée. Dans la page `Détails`, il faut cliquer sur
`Initier le transfert du domaine`. 

Vous pouvez également communiquer le code de transfert au support, qui initiera le transfert du domaine.
