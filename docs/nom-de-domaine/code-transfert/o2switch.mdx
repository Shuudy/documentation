# Récupérer le code de transfert d'un domaine chez o2switch

Ce tutoriel explique comment récupérer le code de transfert, aussi appelé le authcode ou eppcode, d'un nom de domaine
réservé chez o2switch. 

## A quoi sert le code de transfert d'un domaine ?

Le code de transfert peut avoir différents noms. En fonction des registres, vous pouvez retrouver les noms suivants : le
`authcode` (pour authorization code), `authinfo`, ou encore le `eppcode`. Cela représente la même chose à chaque fois.

Ce code de transfert est similaire à un mot de passe, qui est lié à un nom de domaine et qui est connu uniquement du
propriétaire du nom de domaine.

Ce code de transfert permet d'initier un transfert de domaine. C'est un code à fournir lorsque vous souhaitez changer de
registre, donc changer le prestataire qui a la gestion du nom de domaine.

Autrement dit, le code de transfert est nécessaire pour transférer la gestion administrative de votre nom de domaine
d'un registre A vers un registre B.

Le registre entrant, celui qui va récupérer la gestion du nom de domaine, va demander le code de transfert du domaine
pour initier l'opération de transfert. 

Le code de transfert est fourni par le registre sortant, celui qui perd la gestion du domaine. Ce code de transfert est
fourni exclusivement au propriétaire du nom de domaine. 

Le code de transfert est également utile pour réaliser une [demande de
trade](/espace-client/trade/ceder-nom-de-domaine), donc [changer le propriétaire du nom de
domaine](/espace-client/trade/ceder-nom-de-domaine).

## Comment obtenir le code de transfert chez o2switch ?

Pour un nom de domaine réservé chez o2switch, le code de transfert peut se récupérer dans l'espace client.

Le code se retrouve dans `Gérer mes services` puis dans l'onglet `Services actifs` puis en cliquant sur l'icône de
détails en face de la prestation concernée `...`.  Ensuite, il faut cliquer sur `Demander le code de transfert`.

transfert.

Le domaine sera automatiquement déverrouillé afin d'autoriser son transfert sortant vers un autre registre, de votre
choix. 

Le code de transfert sera envoyé par email dés qu'il sera généré. En moyenne, cela prend 3h à être traité.
