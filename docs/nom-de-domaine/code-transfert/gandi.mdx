# Récupérer le code de transfert d'un domaine chez Gandi.net

Ce guide explique comment **récupérer le code de transfert** (authcode) d'un nom de domaine réservé chez Gandi.net.

 Cela permet de transférer la gestion administrative du nom de domaine de Gandi vers **[hébergeur français
o2switch](https://www.o2switch.fr/hebergement-illimite/)**.

Si vous ne savez pas comment fonctionne un transfert de nom de domaine, c'est-à-dire comment transférer la gestion du
domaine d'un registre à un autre, on explique cela plus en détails sur la page [Comment transférer un nom de domaine
?](/nom-de-domaine/transferer-un-domaine).

Dans un premier temps, il faut s'authentifier sur l'interface client de Gandi à l'aide de [ce
lien](https://id.gandi.net/fr/login).

<Badge text="1" /> Une fois connecté sur le tableau de bord, il faut cliquer sur le lien `domaine` dans le menu situé
tout à gauche. 

<Image
    src="/img/docs/nom-de-domaine/code-transfert/gestion-domaine-gandi-1.png"
    title="Menu de gestion des domaines chez Gandi"
    caption="Menu de gestion des domaines chez Gandi"
    alt="Capture d'écran de l'interface de Gandi : Menu de gestion des domaines chez Gandi"
/>

<Badge text="2" /> Ensuite, il faut cliquer sur le nom de domaine pour lequel vous souhaitez récupérer le code de
transfert. 

<Image
    src="/img/docs/nom-de-domaine/code-transfert/preference-domaine-gandi-2.png"
    title="Préférence du domaine Gandi"
    caption="Sélection du domaine dans l'interface de Gandi"
    alt="Capture d'écran de l'interface de Gandi sur les préférences du domaine"
/>

<Badge text="3" /> Sur cette page, il faut aller dans la rubrique `Transfert sortant` puis il faut débloquer le nom de
domaine en cliquant sur `Désactiver la protection contre le transfert`. 

<Badge text="4" /> Ensuite, il faut récupérer le code de transfert (authcode).

    src="/img/docs/nom-de-domaine/code-transfert/recuperer-code-transfert-authcode-gandi.png"
    title="Déblocage et récupération du code de transfert chez Gandi"
    caption="Récupération du code de transfert chez Gandi"
    alt="Capture d'écran de l'interface de Gandi pour récupérer le code de transfert"
/>
