# Récupération des codes de transferts des domaines

Cette page regroupe les procédures et tutoriels pour récupérer les codes de transferts chez les registres les plus 
utilisés en France. 

Les codes de transferts permettent de transférer la gestion administrative d'un domaine vers un autre registre. Par
exemple, cela vous permet :
- de transférer un nom de domaine vers o2switch 
- ou inversement, de transférer un domaine géré chez o2switch vers un autre registre de votre choix 

Si vous souhaitez en apprendre plus sur les procédures de transferts de domaines, vous pouvez consulter : [Transfert de domaine, comment ça marche ?](/nom-de-domaine/transferer-un-domaine)
