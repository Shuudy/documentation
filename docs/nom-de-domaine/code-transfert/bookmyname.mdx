# Récupérer le code de transfert chez bookmyname

Cette page explique comment récupérer le **code de transfert** (authcode) d'un nom de domaine qui est réservé chez
Bookmyname. 

Le code de transfert du domaine permet de transférer la gestion totale du  domaine chez un autre registre, par exemple
vers l'**[hébergeur web o2switch](https://www.o2switch.fr/hebergement-illimite/)**.

Si vous ne savez pas comment fonctionne un transfert de nom de domaine, c'est-à-dire comment transférer la gestion du
domaine d'un registre à un autre, on explique cela plus en détails sur la page [Comment transférer un nom de domaine
?](/nom-de-domaine/transferer-un-domaine).

Pour commencer, connectez-vous sur votre espace chez bookmyname, directement depuis la [page
d'accueil](https://www.bookmyname.com) dans le menu du haut.

<Badge text="1" type="danger" /> Ensuite, naviguez dans la rubrique `Compte` dans le menu supérieur. si vous êtes
connecté, vous pouvez suivre [ce lien](https://www.bookmyname.com/manager.cgi) :

<Image
    src="/img/docs/nom-de-domaine/code-transfert/gerer_bookmyname.png"
    title="Gérer le domaine chez Bookmyname"
    caption="Accès à la gestion des domaines BookMyName"
    alt="Capture d'écran de l'interface de BookMyName pour gérer les domaines"
/>

<Badge text="2" type="danger" /> Ensuite cliquez sur la "liste de mes noms de domaines", ou suivez [ce
lien](https://www.bookmyname.com/manager.cgi?cmd=dld) si vous êtes connecté :


<Image
    src="/img/docs/nom-de-domaine/code-transfert/bookmyname_liste_domaine.png"
    title="Accès à la liste des domaines chez Bookmyname"
    caption="Accès à la liste des noms de domaines gérés chez BookMyName"
    alt="Capture d'écran de l'interface de BookMyName pour accéder aux domaines gérés"
/>

La page suivante devrait afficher la liste des noms de domaines que vous gérez. 

<Badge text="3" type="danger" /> Cliquez sur le nom de domaine pour lequel vous souhaitez récupérer le code de trasnfert

<Image
    src="/img/docs/nom-de-domaine/code-transfert/bookmyname-gestion-domaine.png"
    title="Liste des domaines gérés chez Bookmyname"
    caption="Liste des noms de domaines en gestion chez Bookmyname"
    alt="Capture d'écran de la liste des domaines gérés chez Bookmyname"
/>


<Badge text="4" type="danger" /> Ensuite, cliquez sur "modifier" a côté de "Domaine protégé contre les transferts".

<Image
    src="/img/docs/nom-de-domaine/code-transfert/recuperer-code-transfert-bookmyname.png"
    title="Débloquer le nom de domaine chez BookMyName"
    caption="Il faut débloquer le nom de domaine chez Bookmyname avant de récupérer le code de transfert"
    alt="Capture d'écran de la page d'information d'un domaine bookmyname"
/>

<Badge text="5" type="danger" /> Débloquez le nom de domaine sur la page suivante et validez l'opération. 

<Image
    src="/img/docs/nom-de-domaine/code-transfert/bookmyname-debloquer-domaine.png"
    title="Débloquer le nom de domaine chez BookMyName"
    caption="Il faut débloquer le nom de domaine chez Bookmyname avant de récupérer le code de transfert"
    alt="Capture d'écran de la page d'information d'un domaine bookmyname"
/>

Un message indiquera que le déblocage est en cours. Patientez une ou deux minutes, puis retournez sur la page
d'information du nom de domaine.

La code de transfert s'affichera. 

<Image
    src="/img/docs/nom-de-domaine/code-transfert/code-de-transfert-bookmyname.png"
    title="Le code de transfert s'affiche"
    caption="Après avoir débloqué le domaine, le code de transfert s'affiche"
    alt="Capture d'écran de la page d'information d'un domaine bookmyname"
/>

Vous pouvez maintenant renseigner le code de transfert dans votre espace client o2switch pour lancer le transfert du 
nom de domaine.
