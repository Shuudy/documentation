# Renouvellement de domaine

O2switch vous informe de l'expiration de vos services, incluant les noms de domaines, en envoyant des notifications 
d'expirations par emails, voire par SMS lorsque le numéro renseignée en fiche client noté dans le bon format.

Lorsque le renouvellement automatique n'est pas activé, les premières alertes d'expirations commencent à partir de
J-60. 

Lorsque le renouvellement automatique est activé, les relances entres J-60 et J-15 ne sont pas envoyés. Le 
renouvellement automatique étant planifié à J-15. 

Pour le renouveler, il vous suffit de vous connecter sur votre [interface client](https://clients.o2switch.fr), puis
aller dans la `Gérer mes services`, puis procéder au renouvellement en cliquant sur le lien `Renouveler`.

## Mon domaine est expiré, que faire ?

Si votre nom de domaine est expiré, il faut lancer une procédure de restauration du domaine. Il existe 3 cas possibles :

<Badge text="A" /> Votre domaine est expiré depuis moins d'un mois. 
Normalement cela est rattrapable facilement, vous pouvez restaurer votre nom directement depuis votre interface client.
Si vous avez un doute et que vous souhaitez plus de détails, vous pouvez adresser un email au
[support](https://www.o2switch.fr/support-hebergeur/) pour confirmation.

<Badge text="B" /> Le domaine est expiré depuis plus d'un mois mais moins de 50 jours (entre 30 et 50 jours).

Dans la majorité des cas, cela est rattrapable (il y a quelques exceptions sur certaines extensions de domaines).

Par contre, il y aura des frais de restauration de domaine. Le prix dépend de l'extension de domaine, mais très
souvent c'est facturé bien plus cher qu'un renouvellement classique. Vous pouvez retrouver [la liste des prix sur 
l'espace client](https://clients.o2switch.fr/domaine/prix-domaines).

<Badge text="C" /> Le domaine est expiré depuis plus de 50 jours.

Dans ce cas, votre domaine est de nouveau disponible à l'achat, si personne d'autre ne l'a pris avant vous, vous pouvez
le recommander dans votre interface client comme un nouveau nom de domaine. 

Si vous souhaitez éviter la procédure de récupération à 60€, vous pouvez laisser s'expirer le domaine pour le reprendre
en nom de domaine "classique" immédiatement après mais vous prenez le risque qu'une autre personne réserve le domaine
avant vous. (Si votre domaine est "concurrentiel" et très important, mieux vaut lancer la procédure de restauration
classique). 

## Arnaques sur les domaines

Faites attention aux différents mails (voir même lettre "physique") que vous recevez concernant votre nom de domaine, il
est très peu probable que l'on vous contacte à ce sujet (hormis pour les relances) donc dans le doute, transmettez-nous
le message que vous recevez concernant votre nom de domaine car il existe des arnaques.

Dans le doute, prenez le sujet de votre email ou une partie du message et faites une recherche Google, très souvent vous
aurez la confirmation qu'il s'agit d'une arnaque.
