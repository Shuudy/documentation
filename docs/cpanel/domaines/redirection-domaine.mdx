# Redirections web

L'outil **redirections** de cPanel permet d'ajouter des redirections d'un nom de domaine ou d'une page vers un autre domaine ou, plus spécifiquement, l'une de ses pages.

L'outil de redirections de cPanel peut être utile pour rediriger plusieurs domaines vers un domaine canonique et éviter ainsi le contenu dupliqué.

L'outil peut également être utilisé pour rediriger des anciennes pages d'un site internet (pages supprimées) vers leurs pages équivalentes, cela permet d'éviter d'avoir trop d'erreurs 404 sur son site.

## Créer une redirection simple

Pour créer une **redirection web**, il faut commencer par naviguer dans l'outil **redirection** visible dans la
catégorie des **domaines**.

Ensuite un formulaire de création de redirections apparaît, voici les champs qui apparaissent et qui sont demandés :
  * `Type` : deux choix possibles, 301 pour une redirection permanente (changement d'adresse définitif par exemple) et 302 pour temporaire
  * `http://(www.)?` : définit le domaine qui devra être redirigé. Par défaut, ce sont tous les domaines configurés sur l'hébergement, ce qui est rarement voulu, il faut plutôt sélectionner le domaine que vous souhaitez rediriger
  * `/` : la case vide après le `/` doit être remplie si vous souhaitez rediriger une seule page en particulier du domaine.
Cela représente ce qu'il y a dans la deuxième partie de l'URL.
Si vous ne souhaitez rediriger que la page exemple.com/toto alors il faudra sélectionner le domaine exemple.com et remplir toto dans la case vide après le `/`
  * `Redirige vers` : la page ou le domaine de destination, l'adresse doit commencer par `http://` ou `https://`
  * `Redirection www` : indique quelle version du domaine d'origine doit rediriger. Est-ce le domaine avec www (`www.exemple.com`) ou sans les www devant (`exemple.com`), ou les deux (par défaut, `exemple.com` et `www.exemple.com`)?
  * `Redirection générique` : Permet de rediriger vers la page équivalente du nouveau domaine. Par exemple `exemple.com/toto` va rediriger vers `domaineDestination.tld/toto`.
