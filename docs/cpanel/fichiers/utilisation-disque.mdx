# Contrôle de l'utilisation de l'espace disque

L'outil **utilisation du disque** de l'espace d'hébergement permet de contrôler l'espace disque consommé
par votre [hébergement cPanel](https://www.o2switch.fr/hebergement-illimite/).

L'offre Unique o2switch est un hébergement illimité en espace disque. Cependant l'outil reste utile pour voir quels sont
les gros dossiers sur votre compte.

L'outil peut donc être utilisé pour vous permettre de trouver les dossiers les plus volumineux de votre espace
d'hébergement.

<PreviewToolCard tool='disk_usage' />

## Trouver les dossiers voluminuex

Dans l'outil, vous retrouvez un graphique pour l'utilisation de votre disque.

Les différentes barres de progression ne sont pas créées par rapport à une limite fixe mais par rapport à l'espace
total consommé.

La barre bleu n'indique donc pas une limite, cela indique quels sont les gros dossiers proportionnellement à tout
l'espace consommé sur l'hébergement.

Cela permet de trouver les gros dossiers sur votre espace.
