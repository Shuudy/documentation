# Les outils de gestion des fichiers

Sur cette page, vous retrouvez tous les outils en rapport avec la gestion des fichiers sur [les hébergements mutualisés o2switch](https://www.o2switch.fr/hebergement-illimite/).

<Image
    src="/img/docs/cpanel/fichiers/outils-fichiers.png"
    alt="Capture d'écran des outils de la catégorie Fichier de l'hébergement mutualisé o2switch"
    caption="Capture d'écran de la rubrique des Fichier dans l'interface technique"
    title="Les outils de gestion des Fichiers" />

Dans la catégorie des fichiers, vous retrouvez des outils pour :
  * accéder rapidement à vos fichiers depuis une interface web
  * protéger un répertoire accessible depuis une URL avec un mot de passe
  * contrôler l'utilisation de l'espace disque
  * créer des comptes FTP
  * gérer les projets git avec une interface gitweb
