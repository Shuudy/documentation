# Gestionnaire de fichiers : envoyer et récupérer des fichiers

Le gestionnaire de fichiers de l'[hébergement o2switch](https://www.o2switch.fr/) est un outil permettant de naviguer
et d'éditer les fichiers présents sur votre espace d'hébergement site web.

Cet éditeur est très pratique pour des opérations de maintenance ou d'édition de fichier rapide. Cela permet d'éviter
de lancer et [configurer un client FTP](/guides/configuration-logiciel).

## Présentation du gestionnaire de fichiers

Lorsque vous ouvrez pour la première fois le gestionnaire de fichiers, un menu vous propose de choisir dans quel dossier
vous allez être placé automatique.

Si vous souhaitez modifier les fichiers d'un site internet en particulier, vous pouvez choisir l'adresse du site dans
la liste déroulante, cela vous placera directement à la racine du site en question.

Une option est également présente pour afficher les fichiers cachés (ceux qui commencent par un “.” comme le .htaccess).

Pour retrouver ce menu, vous pouvez cliquer en haut à droite sur les paramètres.

Ensuite, le gestionnaire de fichiers s'ouvre. Ce dernier est organisé en plusieurs parties :

Tout en haut du gestionnaire, vous avez une barre d'outil vous permettant de faire plusieurs actions telles que :
  * compresser/décompresser des archives,
  * envoyer des fichiers,
  * télécharger des fichiers,
  * éditer des fichiers directement depuis le navigateur web : pratique pour éditer rapidement un fichier de configuration,
  * déplacer ou renommer des fichiers
  * supprimer des fichiers

Dans la partie gauche, vous avez toute l'arborescence de votre espace d'hébergement vous permettant de parcourir votre espace et de changer de dossier

Dans la partie droite, vous avez la liste des fichiers présents dans le dossier courant

:::warning
Les opérations de compression et décompression de fichiers doivent être effectuées sur des petits dossiers.

Pour des dossiers plus gros, faites cela directement sur votre ordinateur, n'utilisez pas l'outil de décompression
pour déployer un gros site internet sous forme d'archive, préférez l'utilisation d'un client FTP.

Si vous n'avez pas d'autre solution contactez le support qui peut décompresser des gros dossiers sans que cela ne
ralentisse votre espace d’hébergement.
:::
