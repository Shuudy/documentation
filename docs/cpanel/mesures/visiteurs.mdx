# Logs des 1000 dernières visites

L'outil **visiteurs** de cPanel permet de consulter les 1000 dernières lignes des logs d'accès du serveur web et
ainsi voir les dernières visites de votre site.


<PreviewToolCard tool='visitors' />

## Consulter les logs d'un domaine

Sur la page d'accueil de l'outil, il faut cliquer sur le nom de domaine pour lequel on souhaite consulter les 1000
dernières lignes des logs d'accès du serveur web.

<Image
    src="/img/docs/cpanel/mesures/visitors-accueil.png"
    alt="Capture d'écran de la page d'accueil de l'outil de consultation des 1000 dernières lignes de log"
    caption="Sélection du nom de domaine"
    title="Selection du site internet pour lequel consulter les derniers logs d'accès" />

La page suivante affichera un petit outil de consultation visuel des logs.

<Image
    src="/img/docs/cpanel/mesures/visitors-details.png"
    alt="Capture d'écran de l'outil de consultation de logs"
    caption="Outils de consultation des logs d'accès"
    title="Outil pour consulter les logs" />

L'outil à plusieurs fonctionnalités.

<span className="badge badge--danger">1</span> Dans la zone supérieure droite, la pagination s'affiche.

<span className="badge badge--danger">2</span> L'outil permet de faire une recherche sur un mot clé dans les 1000
dernières lignes de logs. Utile si vous êtes à la recherche d'un élément précis.

<span className="badge badge--danger">3</span> Vous pouvez changer le nombre d'éléments à afficher par page.

<span className="badge badge--danger">4</span> Dans les réglages, accessible en cliquant sur la roue crantée, vous
pouvez afficher les colonnes à afficher ou masquer dans l'outil.

<span className="badge badge--danger">5</span> Vous pouvez forcer l'actualisation de l'outil pour chercher des nouvelles
lignes de logs.
