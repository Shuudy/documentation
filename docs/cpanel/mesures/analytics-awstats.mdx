# Consulter les statistiques de fréquentation avec Awstats

L'outil AWStats génère des statistiques de fréquentation, sous forme de graphique, pour les différents sites que
vous hébergez.

Cela vous permet d'avoir une idée sur le nombre de visiteurs que vous pouvez avoir, l'origine des visiteurs et les pages
les plus consultées sur votre [hébergement internet](https://www.o2switch.fr/hebergement-entreprise/).

<PreviewToolCard tool='awstats' />

## Consulter les statistiques d'un site internet

En naviguant dans l'outil awstats, la première page de l'outil affiche la liste des noms de domaines configurés
sur votre espace d'hébergement.

Awstats fait la distinction, pour chaque domaine, entre :
  * les requêtes effectuées sur la version `http://` du site
  * les requêtes effectuées sur la version `https://` du site

Si votre site est en HTTPS, alors il faut bien cliquer sur la version SSL. L'autre version sera quasi-vide et c'est
normal.

De plus, Awstats utilise le sous-domaine spécial créé par cPanel. C'est pour cette raison qu'un domaine comme
`o2tutofr.dojo2567.odns.fr` peut s'afficher. Il faut l'ignorer, consultez uniquement la colonne des **Domaines liés**.
Tout cela est liée à une spécifitée de cPanel qui n'a pas d'importance dans le cas présent.

Pour consulter les statistiques d'un site, il suffit de cliquer sur le bouton **Voir**.

Ensuite l'outil Awstats s'affiche avec plusieurs graphiques de statistiques et différentes rubriques.

Les statistiques sont générées à partir des logs du serveur web. Donc cela va inclure le trafic de tous les robots,
c'est pour cette raison que vous pouvez constater une différence avec un autre outil comme Google Analytics.

Dans les informations remontées par l'outil, vous pouvez retrouver :
  * un histogramme des visites mensuelles, sur l'année en cours
  * un histogramme des visites journalières, sur le mois en cours (ou un mois précédent si sélectionné dans le menu déroulant)
  * la même chose avec les jours de la semaine
  * et le trafic heure par heure
  * le top 25 du pays d'origine des visiteurs
  * le top 25 des adresses IP ayant le plus fréquenté le site
  * le top 25 des robots ayant le plus fréquenté le site
  * la durée des visites en moyenne
  * le type de fichier consulté
  * le top 25 des pages / URL consultées
  * le top 10 des systèmes d'exploitations (basée sur l'analyse des users agent)
  * le top 10 des navigateurs web (analyse des user agents également)
  * l'origine des connexions au site
  * le détail des codes HTTP retournés dans les réponses du site
