# Les outils exclusifs d'o2switch

Sur cette page, vous retrouvez les outils exclusifs d'[o2switch](https://www.o2switch.fr/) mis en avant.

Ces outils ont été développé afin d'ajouter des fonctionnalités manquantes à cPanel et améliorer l'expérience utilisateur. 

<Image src="/img/docs/cpanel/o2switch/outils-exclusifs-o2switch.png" alt="Capture d'écran du panel des outils exclusifs o2switch" caption="Capture d'écran de la rubrique des outils exclusifs dans l'interface technique" title="Une partie des outils exclusifs o2switch"/>

Les outils permettent par exemple :
  * d'administrer un site WordPress plus facilement 
  * de contrôler plus finements les sécurités en place sur l'héberegment 
  * d'obtenir une instance privée de Redis ou Memcached 
  * d'avoir accès à un pool d'adresses IP pour changer les adresses IP des différents sites hébergés 
  * d'améliorer les performances des sites avec des systèmes de caches
