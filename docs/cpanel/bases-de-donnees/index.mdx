# Les outils de bases de données

Sur cette page, vous retrouvez les différents outils de bases de données sur les plans d'[hébergement de site internet](https://www.o2switch.fr/) o2switch.

<Image
    src="/img/docs/cpanel/bases-de-donnees/outils-bases-de-donnees.png"
    alt="Capture d'écran du panel des outils de bases de données"
    caption="Capture d'écran de la rubrique des bases de données dans l'interface technique"
    title="Les outils de bases de données comme MySQL" />

Dans les outils de **bases de données**, vous pouvez retrouver :
  * un outils pour créer des bases de données MySQL ainsi que des utilisateurs
  * phpmyadmin pour gérer les bases MySQL
  * un outil pour ouvrir l'accès à distance aux bases MySQL
  * un outils pou créer des bases de données PostgreSQL ainsi que les utilisateurs
  * phpphadmin pour gérer les bases PostgreSQL
