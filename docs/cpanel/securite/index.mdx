# Les outils de sécurité

Sur cette page, vous retrouvez les différents outils de sécurité sur l'hébergement [o2switch](https://www.o2switch.fr/).

<Image
    src="/img/docs/cpanel/securite/outils-securite.png"
    alt="Capture d'écran des outils de la catégories des outils de sécurité"
    caption="Capture d'écran de la rubrique des sécurité dans l'interface technique"
    title="Les outils de sécurité" />

Dans les outils de sécurité, vous pouvez retrouver :
  * un outil antivirus pour scanner les fichiers de l'hébergement 
  * un outil pour activer ou désactiver modsecurity, un parefeu applicatif (WAF)
  * un outil pour activer l'authentification à deux facteurs 2FA
  * un outil pour génrer des tokens pour l'API de cPanel 
  * un outil pour vérifier la validité des certificats SSL de l'hébergement 
  * un bloqueur d'adresse IP (ou plage d'IPs)
  * un outil pour gérer les certificats SSL
