# Status des certificats SSL

L'outil **Status SSL/TLS** permet de **vérifier la validité** des certificats SSL installés sur votre hébergement. 

L'outil permet également de vérifier si les noms de domaines configurés sur l'hébergement disposent d'un certificat
SSL valides.
