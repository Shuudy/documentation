# Les outils avancés

Sur cette page, vous retrouvez les outils avancés d'[o2switch](https://www.o2switch.fr/).

<Image
    src="/img/docs/cpanel/outils-avances/outils-avances.png"
    alt="Capture d'écran du panel des outils avancés o2switch"
    caption="Capture d'écran de la rubrique des outils avancés dans l'interface technique"
    title="Une partie des outils avancés o2switch"/>

Les outils permettent par exemple :
  * de créer des tâches planifiées (tâches cron)
  * d'accèder en SSH à l'hébergement depuis un navigateur web avec le terminal web
  * de personnalisées les pages d'erreurs 
  * de modifier le comportement par défaut des répertoires sans index
