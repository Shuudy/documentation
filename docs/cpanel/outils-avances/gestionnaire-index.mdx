# Gestionnaire d'index

Lorsqu'un visiteur accède à un répertoire *(exemple http://monsite.fr/mon-dossier/)* plutôt qu'à une page de votre 
site web, le navigateur affiche généralement **la page d'index** du répertoire. 

Si aucune page d'index n'existe, le navigateur **affiche une liste des fichiers** dans ce répertoire. 

L'outil Gestionnaire d'index permet de définir comment le serveur affiche l'index d'un répertoire spécifique à un
visiteur, ou pour **désactiver** l'affichage de l'index pour un répertoire.
