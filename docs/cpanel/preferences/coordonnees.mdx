# Information de contact cPanel

L'outil `Coordonnées` permet de modifier vos préférences de contacts pour votre hébergement cPanel.

En renseignant votre adresse email dans cet outil, vous pouvez recevoir des notifications provenant de cPanel. Vous 
pouvez recevoir des notifications lorsque : 
* Votre adresse de contact change 
* Vous atteignez votre quota disque *(Ceci n'arrivera jamais les hébergements étant illimité en espace disque)*
* Lorsqu'un certificat SSL de votre hébergement arrive bientôt à expiration 
* Le mot de passe de votre compte d'hébergement change 
* Quelqu'un se connecte à votre compte 
* Une authentification externe est configurée sur votre compte 
* La configuration de l'authentification à deux facteurs change
