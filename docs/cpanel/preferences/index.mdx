# Les outils de personnalisation des préférences

Les outils de la catégorie "Préférences" vous permettent d'ajuster les paramètres de votre hébergement selon vos souhaits.

<Image
    src="/img/docs/cpanel/preferences/outils-preferences.png"
    alt="Captures d'écrans des outils de personnalisations des préférences"
    caption="Les outils de personnalisation des préférences o2switch"
    title="Une partie des outils de personnalisation des préférences"/>

Les outils permettent par exemple :
  * changer les coordonnées de contact pour recevoir des notifications de cPanel
  * modifier la langue de l'interface cPanel
  * changer de mot de passe
