# Mot de passe et sécurité

L'outil `Mot de passe et sécurité` permet de **changer le mot de passe** de votre 
[hébergement cPanel](https://www.o2switch.fr/hebergement-illimite/). 

Après le changement de mot de passe, vous serez déconnecté et il sera nécessaire de vous ré-authentifier.
