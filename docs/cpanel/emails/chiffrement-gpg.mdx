# Chiffrement GPG

L'outil **Chiffrement** vous permet de créer, stocker et gérer les **clés GnuPG**.

Les logiciels de messageries qui utilisent les clés GnuPG chiffrent les messages sortants avec une clé publique.
Seul le destinataire prévu, qui possède la clé privée, peut déchiffrer le message.

À noter, que l'outil **Chiffrement** dans cPanel permet uniquement de créer/stocker les clés, mais cPanel ne va pas
automatiquement chiffrer les emails que vous envoyez.

<PreviewToolCard tool='encryption' />

Le chiffrement doit être géré au niveau du logiciel de messagerie que vous utilisez :
  * [Apple Mail](https://support.apple.com/guide/mail/sign-or-encrypt-emails-mlhlp1180/mac)
  * [Thunderbird](https://support.mozilla.org/en-US/kb/digitally-signing-and-encrypting-messages)
  * [em Client](https://www.emclient.com/blog/pgp--encrypt---sign-emails-in-a-few-clicks-302)
