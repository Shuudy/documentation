# Comportement de l'antispam

L'outil **comportement de l'antispam** vous permet de gérer l'antispam de votre hébergement web en vous laissant la possibilité de :
  * activer ou désactiver le trie automatique des emails dans le dossier spam de l'adresse de messagerie.\
Par défaut, les emails considérés comme spam sont triés dans un dossier spam mais vous pouvez décider de les recevoir en boite de réception (avec un sujet mis à jour avec “SPAM” devant)
  * activer ou désactiver totalement l'antispam, dans ce cas, vous recevez tout en boite de réception sans altération du sujet

<PreviewToolCard tool='apache_spam_assassin' />

## Gérer l'antispam

La gestion de l'antispam se fait très simplement avec deux options :
  * une pour activer ou désactiver le fait de placer automatiquement les messages considérés comme spam dans un sous dossier spam de la boite email concernée
  * l'autre option permet d'activer/désactiver totalement l'antispam de l'hébergement
