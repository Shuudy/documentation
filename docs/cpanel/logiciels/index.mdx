# Les outils de gestions de PHP, NodeJS, Ruby et Python

Sur cette page, vous retrouvez tous les outils permettant de gérer les versions et configurations des langages de
programmation proposés sur les offres d'[hébergements o2switch](https://www.o2switch.fr/hebergement-illimite/) : PHP, NodeJS, Python et Ruby.

<Image
    src="/img/docs/cpanel/fichiers/outils-fichiers.png"
    alt="Capture d'écran des outils de la catégorie des Logiciels de l'hébergement mutualisé o2switch"
    caption="Capture d'écran de la rubrique des Logiciels dans l'interface technique"
    title="Les outils de gestion des Logiciels" />

Dans la catégorie des logiciels, vous retrouvez des outils pour :
  * changer la version de PHP ainsi que la configuration php.ini
  * créer des environnements NodeJS, avec la version de nodeJS de votre choix
  * créer des environnements Python, vous pouvez choisir la version de Python de votre choix
  * créer des environnements Ruby, la version de ruby est également sélectionnable
